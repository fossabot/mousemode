# mousemode
| Command     | Shortcut            |
| :---        | :---                |
| Move Up     | `Shift` + `↑`      |
| Move Down   | `Shift` + `↓`      |
| Move Left   | `Shift` + `←`      |
| Move Right  | `Shift` + `→`      |
| Click Left  | `LShift` + `RShift` |
| Click Right | `A` + `S`           |
